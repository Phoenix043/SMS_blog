Getting Started

GettingStarted.sh 
                        
        # Clone the repository
        git clone https://github.com/Vikram043/Scholario-EduTech-Empowerment-System
        
        # Move to the frontend directory
        cd frontend
        
        # Install necessary packages
        npm install
        
        # Move to the backend directory
        cd ../backend
        
        # Install required Python packages
        pip install -r requirements.txt

Contributing to Scholario

# Fork the Project

First, fork the Scholario project on GitHub by clicking the "Fork" button on the project's repository page. This will create a copy of the project in your GitHub account.


# Create your Feature Branch

After forking, create a new branch in your forked repository. This branch will be used to work on your contributions.


# Commit your Changes

Make the necessary changes, improvements, or bug fixes in your feature branch. Use clear and descriptive commit messages.


# Push to the Branch

Push your changes to the feature branch in your forked repository.


# Open a Pull Request

Once your changes are pushed to your forked repository, open a Pull Request (PR) on the original Scholario repository. Provide a detailed description of your changes in the PR, and our team will review it.

Thank you for contributing to Scholario's development!
